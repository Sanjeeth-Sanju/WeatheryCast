 # **WeatheryCast** - Weather Forecast Web Application
WeatheryCast is a real-time weather forecast website built using **Node.js**, **Express**, **Axios**, and **EJS**.  
Users can enter any city in the world and instantly view temperature, humidity, wind speed, cloudiness, and weather description — all retrieved from the **OpenWeather API**.

# **Features** 
- Real-time weather updates using OpenWeather API  
- Beautiful **glassmorphism UI design**  
- Fully **responsive** across desktop, tablet, and mobile  
- **Error handling** for unsupported locations  
- Dynamic rendering using **EJS templating**

# Tech Stack
- **Backend** - Node.js + Express  
- **Templating** - EJS  
- **HTTP Client** - Axios  
- **API** - OpenWeatherMap (free)   
- **Frontend** - Pure HTML + CSS (no frameworks)  
- **Design** - Glassmorphism (I tried) + CSS Gradients  
- **Deployment** - Render

# **Installation and Setup**

1) Cloning:
   - Clone this repository in your github.
   - _command : git clone https://github.com/Sanjeeth-Sanju/WeatheryCast.git_

2) Install dependencies:
   - You need to install all the required dependencies.
   - __command_ : npm install_

3) Secret Key:
      - Create .env file in the root.
      - API_KEY = "Enter your **OpenWeatherMap** API key"
      - Save the file.

4) Start the server:
     - You need to start the server.
     - _command : nodemon index.js_

5) Open in browser:
      - Run _http://localhost:3000_ in your browser.
